fswatch

This manual is for fswatch (version 1.16.0, 2 June 2021), a cross-platform file change monitor with multiple backends: Apple macOS File System Events, *BSD kqueue, Solaris/Illumos File Events Notification, Linux inotify, Microsoft Windows ReadDirectoryChangesW and a stat()-based backend.
